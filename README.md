# Exchange-data-rt
Create database and Run django web server in port 8000.
$pip install requirement.txt

$python manage.py makemigrations

$python manage.py migrate

$python manage.py runserver 8000

Run nodejs server in port 4000

/nodejs/

$npm install

$node index.js
